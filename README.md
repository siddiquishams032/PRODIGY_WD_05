# Weather App

This is a simple weather application built using HTML, CSS, and JavaScript. It shows real-time weather information for a selected location.

## Features
- Get current weather details
- Search weather by city name
- Displays temperature, humidity, and conditions
- Clean and simple UI

## Tech Used
- HTML
- CSS
- JavaScript
- Weather API

## How to Use
- Enter a city name
- Click search
- View weather details instantly

## How to Run
Open `index.html` in your browser.

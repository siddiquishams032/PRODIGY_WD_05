# 🌦️ Weather Web App

## 📌 Description
This is a weather web application built using HTML, CSS, and JavaScript. It fetches real-time weather data using an API and displays temperature, humidity, and weather conditions.

## 🚀 Features
- Real-time weather updates
- Search by city name
- Displays temperature, humidity, and condition
- Responsive design
- Clean UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Weather API

## 📁 How to Use
1. Enter city name  
2. Click search  
3. View weather details instantly
